
# Using and installing 'Weather Website'


## Run Locally

Clone the project

```bash
  git clone <project-id.git>
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Optional

```bash
Open https://weather-website-lyart.vercel.app/
```
